Demo-safe interface mode
When enabled, LiveWire can obscure sensitive identifiers such as IP addresses, hostnames, usernames, and device-like labels in the UI. This does not modify stored data.
Suggested use: Turn this on before recording walkthroughs, taking screenshots, or showing LiveWire publicly. It is designed to reduce exposed operational intelligence without affecting real telemetry or stored records.

Enhanced Hardware Telemetry

Manage LibreHardwareMonitor integration for richer AMD and Windows sensor visibility.

Best for AMD and richer Windows telemetry
When enabled, LiveWire can use LibreHardwareMonitor on monitored Windows systems to improve GPU, temperature, and hardware sensor visibility beyond standard WMI and native tool output.
Telemetry behavior: NVIDIA systems can continue using native tooling such as nvidia-smi. Enhanced telemetry is especially useful for AMD systems and for richer Windows sensor reporting.
